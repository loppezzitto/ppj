# Primeros Pasos con Jesús

Sitio web devocional construido con React y Vite. El objetivo es acompañar a personas que están comenzando su camino de fe con contenido sencillo sobre devocionales, oración, Biblia y recursos prácticos.

## Stack

- React
- Vite
- React Router
- Framer Motion
- Lucide React

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Secciones

- Inicio
- Devocionales
- Aprender a orar
- Conociendo la Biblia
- Recursos
- Sobre mí
